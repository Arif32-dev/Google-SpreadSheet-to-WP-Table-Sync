==================================== Description ====================================


Display Google Spreadsheet data to WordPress table in just a few clicks and keep the data always synced. Organize and display all your spreadsheet data in your WordPress quickly and effortlessly.

For usage of this plugin watch the full video  https://vimeo.com/526189502

Sheets To WP Table Live Sync is a WordPress plugin with new concept to facilitate anyone showing Spreadsheet data directly to WordPress website with fastest way in just a few clicks. Organize and display all your spreadsheet data in your WordPress quickly and effortlessly. Order up all your tables in the simplest way possible.

The plugin supports sorting options, search bars, and swap options. Keep almost every kind of data safe and secure and manage them smoothly without any hassle. Sheets To WP Table Live Sync plugin creates responsive data tables and makes powerful data administration synchronization. Anyone will be able to handle it even without any programming knowledge. 

It’s easy to install and will offer you the simplest settings to configure. Even a beginner can create a table on the backend by exporting data from any google sheet. After you set up a table, a shortcode is generated automatically. All you need to place the shortcode anywhere you want on your website. All table data is displayed in a spreadsheet format in your dashboard. It doesn’t matter what type of data you would like to include, Sheets To WP Table Live Sync plugin has support for almost everything you can think of. 



==================================== Free Features ====================================


* Live data synchronization: This is the most attractive feature of Sheets To WP Table Live Sync plugin. The plugin will automatically synchronize all the data. Your website’s data table will be updated by exporting all your data from the table or your spreadsheet.

* Shortcode supported: Place your data table more easily and effortlessly anywhere with the shortcode feature. A shortcode will be created with the table created. You can place the code anywhere you want on your page or posts. The table will appear according to your order.

* Table title: Enable this to show the table title in the h3 tag above the table in your website front-end. Rename your tables as you want. You will get an option to turn it On or Off.

* 15 rows fetching from the sheet: Create a table and fetch up to 15-row data from your targeted table. You can always get a chance to add 15 rows maximum in a single table. This will show rows per page in the frontend.

* Show Info block: Show Showing X to Y of Z entries block below the table. With this feature, you will get the idea of the data sequence of your table at a glance. The plugin also has an On/Off button on the backend for this feature. 

* Show X entries: This option appears on the top of your table. It counts your total entries on that page of your table. This feature also contains an On/Off button on the backend.

* Swap filters: With this swap filtering feature you can swap the filtering positions from left to right or right to left.

* Swap bottom elements: Swap elements from top to button or bottom to top with Sheets To WP Table Live Sync plugin. Swap the places of Showing X to Y of Z entries with table pagination filter

* Allow sorting: Sort all your elements with Sheets To WP Table Live Sync plugin. It allows data sorting on your table in the exact order you needt. You can turn off the feature if you want your data not sorted.

* Search bar: Find your data and related details specifically with the search bar. The search bar will help you to find any data easily and effortlessly. This feature also contains an On/Off button on the backend.

* Pagination Support: The Sheets To WP Table Live Sync plugin plugin comes with the pagination support option. Order your data with a simple manner of sequential pages which are contextually connected

* Asynchronous loading: Enable this feature to load the table in the frontend after loading all content with a preloader. This will help your website load fast. Otherwise, the table will load with the reloading of the browser every time.

* Gutenberg Block: Sheets To WP Table Live Sync also supports Gutenberg blocks. You’ll be able to manage your tables with the Gutenberg block anytime, every time.

* 24/7 Support: Sheets To WP Table Live Sync got the back of a dedicated hardworking team. Get support anytime in any issues with Sheets To WP Table Live Sync plugin. Our support team will be there always.


==================================== Pro Features ====================================


* Responsive Table: Sheets To WP Table Live Sync plugin is responsive for any device. The plugin allows collapsing on mobile and tablet screens.

* Export table: Table Exporting via CSV, Excel, PDF, JSON, Print, Table Copy is easy on this plugin

* 50 rows fetching from the sheet: Fetch up to 50 row data on this feature.

* Elementor Widget Support: Sheets To WP Table Live Sync supports elementor widget. Organize your table data effortlessly than ever.
